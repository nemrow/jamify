== Jamify!

== Servers

== Staging Server: 
http://jamify-staging.herokuapp.com<br><br>

To push to staging server:<br>
$ git push staging master<br>
To migrate staging server:<br>
$ heroku run rake db:migrate --remote staging<br><br>

== Production Server:
http://jamify.herokuapp.com

To push to production server:
$ git push production master
To migrate production server:
$ heroku run rake db:migrate --remote production
